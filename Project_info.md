## How to Open This Project

1. Download Sales_dashboard.pbix
2. Open with Power BI Desktop (free):
   https://powerbi.microsoft.com/downloads
3. All data is embedded — no external source needed

## Dataset
AdventureWorks — Microsoft Sample Database
Tables: Sales, Product, Customer, Calendar, Territories, Budget

## Tools Required
- Power BI Desktop (any version 2022 or later)
